Requirements
------------

Package requires PHP 5.3.7 or higher

- [nette/utils](https://github.com/nette/utils)

Installation
------------

The best way to install XRuff/CsvParser is using  [Composer](http://getcomposer.org/):

```sh
$ composer require xruff/csvparser:dev-master
```
